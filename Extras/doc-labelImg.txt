INSTALACION
1. Crear entorno virtual --> python -m venv venv
2. Correr entorno virtual --> source venv/bin/activate
3. Instalar dependencias: pip install pyqt5==5.15.2 lxml
4. Clonar repositorio de labelImg --> git clone https://github.com/heartexlabs/labelImg.git
5. Compilar qt5py3 --> cd labelImg && make qt5py3

USO
1. Correr labelImg --> python labelImg.py
2. Abrir directorio donde estan guardadas las imagenes --> Open Dir > Tensoflow/workspace/images/collected_images
3. Una por una etiquetar las imagenes:
    3.1. View > Check "Auto Save Mode"
    3.2. Presionar "w" para activar el modo "RectBox"
    3.3. Dibujar marco alrededor del gesto
    3.4. Aparecerá un pop-up en el cual debo ingresar el nombre de la etiqueta deseado
    3.5. Presionar Ok
    3.6. Presionar "d" para pasar a la siguiente imagen
    3.7. Repetir para todas las imagenes.
Esto generará un archivo .xml para cada imagen
